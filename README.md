# deepseek-ai 

 "deepseek-ai" VS code Extension provides chat like interface to chat with deep seek AI. 

# Requirements

1. Install Ollama

>https://ollama.com/download/windows

2. Install and run deepseek-r1:7b model

> ollama run deepseek-r1:7b
